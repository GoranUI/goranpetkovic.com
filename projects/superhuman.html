Overview

BioAge is a health platform that reveals the gap between chronological and biological age using blood biometrics and genetic analysis, empowering users to make informed lifestyle choices for longevity.

Project brief

Design a web platform from zero that translates complex biomarker and DNA data into personalized, actionable health insights while building an intuitive admin system for managing patient data.

Role

As a sole product designer, I led discovery workshops, information architecture, user flows, wireframing, and end-to-end high-fidelity UI design, delivered within a 3-month deadline.

Methods

  • Collaborative discovery via Miro and FigJam workshops.
  • Full admin and user flow mapping before UI.
  • Advocated for custom data visualizations over Chart.js frameworks to ensure clarity.
  • Used a modal-based admin system to minimize complexity while maintaining design consistency.
