Overview

StructAI is an enterprise-grade construction configurator combining a precision CAD canvas, AI-guided structural analysis, real-time compliance checking, and team collaboration, collapsing weeks of back-and-forth between architects, engineers, and contractors into a single intelligent workflow.

Project brief

Design a professional desktop-first platform that feels as precise as CAD software but as intuitive as modern SaaS, supporting multiple user types across complex multi-level construction projects.

Role

As a sole product designer, I oversaw end-to-end UX/UI, including CAD canvas, AI analysis visualization, design system, team management, reporting, and full workflow architecture.

Methods

  • Stakeholder workshops mapping workflows across architects, engineers, and project managers.
  • Competitive analysis of CAD tools and SaaS configurators.
  • Information architecture for multi-panel layout.
  • User flow mapping across 3 user types.
  • Design system built from scratch, covering typography, color, components, and states.
